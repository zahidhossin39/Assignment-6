Name : DigiTools / Assignment-6

Description : I created this website so people can buy the tool subscription easily, so they don't have to go to different websites to buy those packages. They just need to come here and buy whatever tool they want. 

Technology that I use : 
1. I use daisy UI and tailwind.css to make the UI design.
2. I use JavaScript to make it responsive and also use JSON.
3. I use JSON that I created using an AI and then use it in my website.
4. Obviously, I use React JS to make it overall much easier.
5. I use the react-toastify to show the alerts.


3 Features about my project : 
1. Interactive product with shopping cart functionality
2. The three-step onboarding process
3. Transparent pricing plan.
